# Campanha Ecos da Serra

Painel local em HTML/CSS/JS para organizar a campanha investigativa **Ecos da Serra**, ambientada em São Sebastião da Serra.

## Como abrir

Abra o arquivo `index.html` em qualquer navegador moderno.

## Como publicar no GitHub Pages

Este projeto é um site estático de arquivo único. Para publicar, basta subir estes arquivos para um repositório no GitHub e ativar o GitHub Pages na branch principal.

## Observação sobre notas

As notas digitadas nos campos do painel são salvas no navegador de cada pessoa usando `localStorage`. Elas não são sincronizadas entre jogadores e não ficam salvas no GitHub.
